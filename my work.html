My Work

Project 1: I created a simple, single-page website dedicated to the character Killua Zoldyck from the anime *Hunter x Hunter*. The website features a clean, minimalist design to highlight Killua's key traits and journey. It includes a brief biography, showcasing his background as a skilled assassin and his growth throughout the series. The color scheme and imagery are inspired by Killua’s character. Overall, the website provides a concise and engaging tribute to Killua in a user-friendly format. However, it’s not perfect, as this was my first time coding a website.

Project 2: I created a single-page website for West Visayas State University Lambunao, showcasing its vision of becoming a leader in higher education. The site emphasizes the mission to provide accessible education, promote research, and engage in community service. Core values such as integrity, excellence, innovation, and social responsibility are highlighted. The design is clean and user-friendly, ensuring easy navigation while presenting the university's goals. This website fosters an emotional connection with students, faculty, and the local community, reflecting the university’s academic achievements and digital identity.

Project 3: I made a simple weather app website, although it is not fully functioning yet. The website is responsive, ensuring it works well on different devices. I faced some struggles with the code, but I did my best to overcome them. Despite the challenges, I was able to create a clean and user-friendly design. The app was built using HTML,and CSS,. While it's not fully operational, it was a great learning experience. I'm proud of the progress I made in developing this app.

Project 4: This single-page website is a simple and stylish design centered around coffee culture. The header features a clean navigation bar with links to Home, About, and Contact. The main section displays an eye-catching image of various coffee mugs arranged in a circle, paired with the phrase "Coffee O'Clock" to convey a playful theme. A detailed description of a latte, accompanied by an elegant coffee cup, highlights the focus on premium coffee beverages. Social media icons at the bottom add an interactive element, connecting users to additional platforms.
